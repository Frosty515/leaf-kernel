# Leaf Kernel
A x86_64 kernel.

## Building
To build an ISO. Just simply run:
```bash
make
```

## Requirements
- nasm
- Crosscompiler and binutils for target (x86_64-elf)
- xorriso

## Architectures
| Architecture 	| Status 	|
|--------------	|--------	|
| x86-64       	| ✅        |
| x86          	| ❌        |
